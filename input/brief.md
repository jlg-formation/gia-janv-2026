# Brief : Projet de TP sur le RAG (Retrieval-Augmented Generation)

## Objectif

Ce projet de travaux pratiques vise à explorer et implémenter une architecture RAG complète.
Le RAG combine la puissance des modèles de langage avec une base de connaissances externe.
L'objectif est de créer un système capable de répondre à des questions en s'appuyant sur des documents.

## Fonctionnalités attendues

Le système devra indexer des documents textuels dans une base de données vectorielle.
Il devra rechercher les passages pertinents en fonction d'une requête utilisateur.
Les résultats seront ensuite transmis à un LLM pour générer une réponse contextualisée.

## Livrables

Un prototype fonctionnel démontrant le pipeline complet : ingestion, recherche et génération.
